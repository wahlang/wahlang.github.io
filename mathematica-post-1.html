Let us draw a block sliding an inclined plane which is an example in physics where we draw a free body diagrams indicating the direction of different forces acting on the block. Here is how you do it:

Code that generate figure 1. 

                        
    g1 = Graphics[Line[{{0, 0}, {30, 0}}]];    
    g2 = Graphics[{Thick, Line[{{0, 0}, {20, 20}}]}];
    g3 = Graphics[{Opacity[0.2], Blue,Rotate[Rectangle[{8, 8}, {12, 12}], 45 Degree, {Left, Bottom}]}];
    g4 = Graphics[{Orange, Thickness[0.01], Arrow[{{8, 11}, {8, 2.0}}]}];
    g5 = Graphics[{Cyan, Thickness[0.01], Arrow[{{8, 10.8}, {3.8, 15}}]}];
    g6 = Graphics[{Red, Thickness[0.008],Arrow[{{2.4, 5.2}, {6.6, 9.4}}]}];
    g7 = Graphics[Text[StyleForm["Weight", FontSize -> 14, FontWeight -> Bold], {10, 5}, {0, 1}, {0, -1}]];
    g8 = Graphics[Text[StyleForm["Normal", FontSize -> 14, FontWeight -> Bold], {9,12.2}, {1, 1}, {1, -1}]];
    g9 = Graphics[Text[StyleForm["Friction", FontSize -> 14, FontWeight -> Bold], {3, 8}, {0, 1}, {1, 1}]];

        (*Finally*)
        Show[g1, g2, g3, g4, g5, g6, g7, g8, g9]
