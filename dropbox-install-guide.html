How to install Dropbox in ubuntu or any other linux Operating

Published blog-list13:12..

Dropbox is a file hosting service operated by the American company Dropbox, Inc., headquartered in San Francisco, California, that offers cloud storage, file synchronization, personal cloud, and client software. Thus, this is one of the software which is a must have in your PC if you are doing research and have collaborators where you need to share your project files. They will provide you 2 Gb of free storage and unfortunately if you need more than this you will have to pay. I've been using the free version without any issue.

Here is how you install it:

Via the terminal or command

The Dropbox daemon works fine on all 32-bit and 64-bit Linux servers. To install, run the following command in your Linux terminal.

For 32 Bit:

cd ~ && wget -O - "https://www.dropbox.com/download?plat=lnx.x86" | tar xzf -


For 64 Bit:

cd ~ && wget -O - "https://www.dropbox.com/download?plat=lnx.x86_64" | tar xzf -


Next, open your terminal, copy and execute the following line of command.

~/.dropbox-dist/dropboxd


You might need to do this everytime you wan to run dropbox if you don't know how to do automation. You can get more info at https://www.dropbox.com/install. Enjoy!
